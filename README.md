# Моё портфолио

Простой адаптивный сайт-портфолио фотографа.

## Что есть на сайте

- слайдер на 3 изображения;
- кнопки переключения слайдов;
- индикаторы слайдов;
- автопереключение каждые 5 секунд;
- блок "Обо мне";
- блок "Навыки";
- галерея из 12 фотографий;
- адаптация под телефон.

## Файлы проекта

- `index.html` — структура страницы;
- `style.css` — стили и адаптивность;
- `script.js` — логика слайдера;
- `images/` — локальные изображения.
